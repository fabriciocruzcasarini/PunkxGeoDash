# 📍 GeoDash SQLite

Dashboard para gerenciamento de coordenadas geográficas com mapa interativo.

## 🚀 Tecnologias
- PHP 7.4+
- SQLite
- Leaflet.js (Mapas)
- Bootstrap 5 & jQuery

## 🔧 Instalação
1. Suba os arquivos para um servidor com suporte a PHP.
2. Certifique-se de que a pasta tenha permissão de escrita para a criação do banco `.db`.
3. Acesse `index.php` e use o botão "Importar" para carregar os dados iniciais.
